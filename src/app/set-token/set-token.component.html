Az oldal használatához meg kell adnod egy GitHub access tokent. Ennek megadásához jelentkezz be a GitHub felhasználói fiókodba, majd a jobb felső sarokban található lenyíló menüben válaszd a Settings opciót. A bal oldali sávból válaszd a Developer settings menüpontot, majd a Personal access tokens-t. Nyomd meg a Generate new token gombot, majd másold be ide a kapott tokent. A mentés gombra kattintva az oldal működésre készen áll. Ha szükséges, a tokenedet bármikor átírhatod.
Ha valamilyen okból törölni szeretnéd a tokenedet, akkor a token törlése gomb segítségével megteheted. Vigyázat, ez a művelet nem visszavonható és a token törlését követően nem leszel képes használni az oldalt.
